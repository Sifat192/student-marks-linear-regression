# Student Marks Prediction (Linear Regression)

## Overview
This project predicts student exam marks based on study time and number of courses using Linear Regression.

## Technologies
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Workflow
1. Data Exploration
2. Model Training
3. Train-Test Split
4. Evaluation

## Result

The Linear Regression model was trained to predict student marks based on study-related features.

**Model Performance:**

- Mean Absolute Error (MAE): **3.26**
- Mean Squared Error (MSE): **12.97**
- Intercept (Baseline Prediction): **24.81**
